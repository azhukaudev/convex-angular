How this demo works

Both buttons open the same todo detail query. The difference is that Prewarm + Open starts a temporary background subscription first, then navigates after a short lead time.

For the clearest comparison, turn on browser network throttling such as Slow 3G before trying both buttons. This page still uses injectQuery() for the real detail view.
