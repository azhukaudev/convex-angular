How to read this demo

This page uses a dedicated paginated dataset sorted by ascending rank. Each action button triggers a mutation wired to a different optimistic pagination helper.

Switch lanes to confirm argsToMatch keeps optimistic changes scoped to the active query. Use Load More until the list is exhausted before trying insertAtBottomIfLoaded().

Try this sequence

  1. Reset the current lane to the baseline ranks.
  2. Run Insert At Top before loading more pages.
  3. Run Insert At Position and watch it slot into the correct page.
  4. Load all pages, then run Insert At Bottom If Loaded.
  5. Toggle any row to see the in-place optimistic update helper.
